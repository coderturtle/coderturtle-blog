# Coderturtle.io Brief

Coderturtle.io is a public website/blog deliverable for the coderturtle identity. It is maintained as a Hekton factory-output project.

The repo is an Astro/Tailwind/MDX site. Production runs on the Hekton AWS static-site pattern (private S3, CloudFront, ACM, Route 53, GitHub Actions OIDC), live and verified as of 2026-07-03. The legacy GitHub Pages workflow is now turned off.
