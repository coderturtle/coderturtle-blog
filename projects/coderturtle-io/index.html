About this project

This is the project behind the site itself: the gateway, the workshop homepage, the design system, and — starting here — the projects-and-build-logs model that replaced the old blog.

Nothing in this build log is staged after the fact. It’s the actual sequence of decisions, dead ends, and fixes from building a site with an AI agent in the loop, in the order they happened.
