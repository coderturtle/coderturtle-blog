- Hekton root: `~/hekton`
- Development root: `~/Development/hekton`
- Project root: `~/Development/hekton/factory-output/coderturtle.io`
- Vault root: `~/vaults/hekton-mind-palace`
- Proposed vault control plane: `20-projects/factory-output/coderturtle.io`

- `git`
- `bash`
- `node`
- `npm`

## GitHub Account

Use the `coderturtle` GitHub account for this repo.

The local remote should use:

```text
git@github.com-coderturtle:coderturtle/coderturtle-blog.git
```

That host alias routes SSH to the coderturtle key according to the Hekton command reference.
